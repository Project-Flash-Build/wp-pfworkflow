# WP-PFWorkflow™

**The visual workflow engine for WordPress.** Build automations as diagrams you can open and read — triggers, conditions, actions and a real execution engine with queue, retries, replay, timeline and idempotency — running inside your own WordPress install, with no external SaaS.

WP-PFWorkflow listens to your site's events as native triggers (WooCommerce orders, record changes, content events, schedules, webhooks and more), routes and orchestrates the steps that follow, and reaches beyond WordPress to your own machine through the open-source [WP-Executor](https://github.com/Project-Flash-Build/wp-executor) runner. It is the engine the [WP-PFAgent](https://github.com/Project-Flash-Build/wp-pfagent) AI agent drives from natural language, and it pairs with the WP-PFManagement™ low-code platform to turn data and process into working apps.

WP-PFWorkflow is a **proprietary, per-customer-licensed WordPress plugin**. The standard build ships obfuscated and is refundable; an optional annual add-on delivers the clean PHP source. It is sold and licensed — not distributed here.
